Core Trading Game — The Arena

Every stock and asset is a trading card with real stats: price, volatility, dividend yield, sector, momentum score, and analyst consensus. Cards have rarity tiers from Common (penny stocks) to Legendary (market movers like AAPL, TSLA, BRK). Trading is done by playing cards onto a board — inspired by Marvel Snap's lane system. Three lanes represent time horizons. Each lane has a capacity limit enforcing portfolio concentration rules. You "play" a card by committing capital to that position. AI agent advisors can suggest plays, but you always confirm — consent-first.

Play Modes — Multiple Ways to Trade

🎰Scratcher Mode — Instant Play
Pick a theme (Tech, Energy, Pharma), scratch to reveal 3 stocks, the AI picks the best micro-trade. As easy as a lottery scratcher. No deck building, no strategy — just scratch and win (or learn).

🃏Arena Mode — The Full Card Game
Build your deck, play the board, compete on leaderboards. Marvel Snap meets Wall Street. Use card abilities strategically. Manage your energy each turn. Outplay the market — and other traders. Every trade is a card played, every position a strategic choice.
🕹️Arcade Mode — Quick Mini-Games
HTML5 Canvas arcade games where your score translates to micro-trades. Breakout where bricks are stocks. Snake where food is dividends. The spirit of Flash-era web games, built with modern tech. Quick, fun, low-stakes — and every game teaches market intuition.
🧪Simulation Mode — Paper Trading Sandbox
Practice with fake money, test strategies, no risk. Full AI analysis of your decisions after each session. Learn without losing. The sandbox is your training ground before you enter the Arena for real.
⚔️Ranked Mode — Compete Against Traders
Seasonal rankings, ELO-style matchmaking based on portfolio performance. Climb the ladder from Bronze to Diamond. Season rewards include exclusive card skins, board themes, and creature accessories. Your skills are your rank.
📊Idle Mode — Set It and Forget It
Set your strategy and walk away. Your AI agents manage positions within the boundaries you define. Check back for loot boxes and progress. The stock market IS an idle game — set up your compound engine and let time do the work.

Genesis Lab — Creature Breeding

Each user gets a Genesis Creature — a unique trading companion born from your first trade. Creature stats derive from trading behavior: aggressive traders get fiery creatures, patient holders get wise ancients, diversified traders get shapeshifters. Creatures evolve based on portfolio milestones and provide passive UI buffs: reduced fees, enhanced predictions, bonus loot chance. Purely cosmetic/gamification — does NOT affect actual trading outcomes.

Loot Boxes & Reward System

All loot is earned, never purchased with real money. This is FERROS — free and open source. Loot tables are transparent and published in the source code. No dark patterns.

📦Trade Milestone Box
Earned every N completed trades. More trades = more boxes.
💎Diamond Hands Box
Earned for holding a position for 30 / 90 / 180 / 365 days.
🔥Streak Box
Earned for consecutive daily logins. Streak multiplier increases quality.
📈Profit Box
Earned when a position hits a profit target you set.
🏆Season Box
Earned at end of ranked season based on your final tier.
🎁Daily Login Reward
Small reward every day, scaling with consecutive days. Day 7 = guaranteed Rare.

Technical Architecture

All data local-first, encrypted at rest, no cloud required, AI runs locally when possible. Smart contract boundaries follow the ADR-002 pattern. Click any layer to expand details.

Core Trading Stack
🔑 FERROS Kernel
Memory-safe foundation, capability-based security
RustCore
The FERROS microkernel provides the security foundation. All trading operations run in isolated capability-scoped processes.
✅ Consent Engine
Trade Window pattern — every action requires explicit approval
Security
Implements the Trade Window consent model. No trade executes without user confirmation. Agents propose, humans approve.
📊 Trading Engine
Order management, position tracking, P&L calculation
EngineReal-time
Manages all trading operations: order placement, position lifecycle, profit/loss tracking, and portfolio state.
🃏 Card System
Asset → Card mapping, abilities, rarity, evolution, deck management
Gamification
Maps financial assets to trading cards. Manages rarity tiers, keyword abilities, card evolution, and deck composition rules.
🖥️ UI Layer
Self-contained HTML/CSS/JS, game board, card rendering
Frontendfile://
Single-file HTML prototype. Zero external dependencies. Works on file:// protocol. All CSS and JS inline.
AI Agent Pipeline
🤖 AI Agent Host
Agent lifecycle, capability grants, trust scoring
Agent
Hosts Scout, Strategist, Guardian, Oracle, and Broker agents. Manages trust levels and capability boundaries.
🧠 Strategy Engine
Deck recommendations, market analysis, pattern recognition
AI
Runs local ML models for pattern analysis. Provides deck optimization and trade recommendations.
🛡️ Risk Manager
Exposure limits, stop-loss automation, diversification enforcement
Safety
Guardian agent backend. Enforces user-defined risk boundaries. Can veto proposed trades that exceed limits.
⚙️ Execution Bridge
Order routing, fill management, timing optimization
Exec
Bridges consent-approved trades to execution. Optimizes entry/exit timing within user-defined parameters.
Data & Progression Layer
🔗 Seal Chain
Tamper-evident trade history, SHA-256 with djb2 fallback
Crypto
Every trade sealed into an append-only chain. SHA-256 primary with djb2 fallback for file:// protocol compatibility.
🎟️ NFT Minting Bridge
Consent-gated on-chain ticket generation
Optional
Optional bridge to mint sealed trades as on-chain NFT tickets. Fully consent-gated. Open source minting code.
🎮 Loot & Idle Engine
Reward calculator, idle progression, prestige system
GameXP
Tracks loot box triggers, idle earnings, prestige multipliers, and creature evolution milestones. All progression client-side.
