## Entry criteria

## Required evidence

The following must all be true and verifiable before G3 closes:

**S4 Runtime:**
- [x] `crates/ferros-core/` builds and passes `cargo test`.
- [x] `crates/ferros-runtime/` builds and passes `cargo test`.
- [x] Policy engine: 10+ unit tests covering grant/deny scenarios.
- [ ] Property tests for policy engine (e.g., via `proptest` or `quickcheck`).
- [ ] `crates/ferros-core/` compiles with `--no-default-features` (validates `no_std` readiness).

**S3 Agent Center:**
- [x] `crates/ferros-agents/` builds and passes `cargo test`.
- [x] `echo` reference agent: registers, receives grant, spawns, echoes, stops.
- [x] `timer` reference agent: registers, spawns, fires periodic events, stops.
- [ ] `ferros agent list | describe | run | stop | logs` all functional.

**Joint (S3 + S4 integration):**
- [x] `ferros-node demo` runs deterministically: starts runtime, registers both reference agents, verifies deny-by-default, exits 0.
- [x] Deny-by-default verified: ungranted capability request from either reference agent is blocked and logged.
- [x] The `echo` and `timer` agents use real `CapabilityGrant` types from S2 (no stubs).
