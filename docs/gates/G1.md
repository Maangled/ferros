- [ ] `Cargo.toml` workspace root exists with `[workspace]` members list.
- [ ] `rust-toolchain.toml` pins a stable Rust edition.
- [ ] `cargo build` exits 0 on `ubuntu-latest`, `macos-latest`, `windows-latest` in CI.
- [ ] `cargo test` exits 0 on all three platforms (empty test suite is acceptable).

- [ ] `cargo clippy -- -D warnings` exits 0.

- [ ] `README.md` reflects actual project state (no aspirational progress bars).
- [ ] `CODEOWNERS` exists with at least per-stream path ownership stubs.
